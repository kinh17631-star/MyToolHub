🛠️ MyToolHub: Manual Installation

Aap niche di gayi commands ka use karke tools ko manually setup kar sakte hain:

1. Update & Python Install

pkg install python git -y

2. Clone and Enter IP-Tracer

git clone https://github.com/kinh17631-star/MyToolHub
cd MyToolHub/IP-Tracer

3. Launch Tool

python3 install.py && python3 main.py
